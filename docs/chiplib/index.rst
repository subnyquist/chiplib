**Chiplib** is a collection of reusable hardware IP blocks implemented in
synthesizable SystemVerilog.

Library Components
==================

:doc:`chiplib_riscv_plic`
  Scalable RISC-V platform-level interrupt controller (PLIC) with a pipelined
  interrupt router, supporting arbitrary source-to-target interrupt mapping.
