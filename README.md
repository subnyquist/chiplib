**Chiplib** is a collection of reusable hardware IP blocks implemented in
SystemVerilog.

## Bug Reports

Bug reports are welcome. Please use the [issue
tracker](https://github.com/subnyquist/chiplib/issues) to file bug reports,
feature requests, queries, and suggestions.
